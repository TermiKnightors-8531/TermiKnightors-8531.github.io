VSCode Introduction

What is VSCode?

VSCode is a place for us to write code. It contains all of the tools that we need to write robot code, to interface with Git and do all of the work that we need to program.

Your first code

You can open VSCode from your desktop. First we'll start with cloning a repository When VSCode has opened, the side bar will have numerous options. Click the one that looks like a spaghetti noodle with meatballs (that's a technical term).
Then click the clone repo button.
At the top, it will ask you to paste a url. Provide the link to your desired github repository. For now, we'll use: https://github.com/TermiKnightors-8531/SparkTests. Once you have pasted this, hit enter. It will prompt you for a folder; place it in a place where you can find it. Perhaps create an FRC folder. Once you follow all the prompts, you can open the repository in your current window. When VSCode asks you if you would like to open the new repo, click open! Congrats Git may ask for verification, but I don't know what that looks like. You programmers are smart! You can figure it out and if not, let your programming head know!

Now, for a sucky part: setting your username and email.

This isn't that bad, but it's a bit of terminal work. To open the terminal in VSCode, press the keys, CTL+SHIFT+`. This will open a terminal.

Click the down-arrow button in the terminal and select git bash. First, paste git config --global user.name "FIRSTNAME LASTNAME" into the git terminal, changing the first and last name as appropriate. Hit enter to submit. Similarly, paste git config --global user.email "email or fake email" into the terminal. You don't need to provide your email. You can just enter random letters if you want. We know where to contact you if you screw something up :). Make sure to keep the quotation marks. They are NECESARRY

You're ready to go!
